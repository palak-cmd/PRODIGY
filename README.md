# Prodigy InfoTech ML Internship
This repository contains the projects I completed during my Data Science internship with Prodigy InfoTech.  
I worked on different datasets and practiced data cleaning, analysis, visualization, and basic machine learning concepts using Python.

## Tasks Completed

### Task 01 - Data Visualization

In this task, I created visualizations to understand the distribution and patterns present in the dataset. It helped me learn how charts can make data easier to understand.

### Task 02 - Exploratory Data Analysis

For this task, I performed data cleaning and exploratory data analysis. I checked the dataset, handled missing values, explored different columns, and used graphs to find useful insights.

### Task 04 - Sentiment Analysis

In this task, I analyzed sentiment-based data to understand public opinion. I used different visualizations to compare sentiments and identify trends in the data.

### Task 05 - Traffic Accident Analysis

For this task, I analyzed traffic accident data and explored accident patterns based on different factors such as time, weather, road conditions, and location-related details.
